# Data Engineering Zoomcamp 2026

This repository contains my solutions for the [Data Engineering Zoomcamp](https://github.com/DataTalksClub/data-engineering-zoomcamp) by [DataTalks.Club](https://datatalks.club/).

## About the Course

The Data Engineering Zoomcamp is a free 9-week course covering data engineering fundamentals. It provides hands-on experience building end-to-end data pipelines using industry-standard tools.

**2026 Cohort Start Date:** January 12, 2026
