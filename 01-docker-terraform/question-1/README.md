# Solution

Create simple image on `Dockerfile`.

Run following commands

```bash
docker build -t de-zoomcamp-01-01:latest . 
docker run --rm -it de-zoomcamp-01-01:latest
```

Then, in the container, run:

```bash
pip --version
# output => pip 25.3 from /usr/local/lib/python3.13/site-packages/pip (python 3.13) 
```

Anwser: pip version 25.3
